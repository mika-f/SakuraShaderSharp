# Motion Graphics Shader by Sakura Shader

モーショングラフィックスシェーダーです。
Quad に図形をいっぱい描画したり、図形を組み合わせたり、それらをアニメーションさせたり出来ます。


## 使い方

適当な Material に `NatsunekoLaboratory/Sakura Shader/Motion Graphics` を割り当てて下さい。
シェーダープロパティの解説については、 GitHub を確認して下さい。

https://github.com/mika-f/SakuraShaderSharp/blob/main/docs/motion-graphics.md


## テスト用アニメーションについて

こちらの動画のモーショングラフィックスをお借りしています。

面白い＆参考になるので、良かったら他の動画も見ていくと参考になると思います。
